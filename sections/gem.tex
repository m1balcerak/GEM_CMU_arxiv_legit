\section{Graph Energy Matching}
\label{sec:gem}
Graph generation requires handling of discrete variables and permutation invariance. Within this work, we thus introduce a permutation-invariant minibatch OT and a multi-regime stochastic sampling strategy together with a JKO-Style Variational Objective for Discrete Probability Spaces that allows us to extend Energy Matching to graphs.

\paragraph{Graph representation.}
We consider a discrete state space $\mathcal{X}:=\big(\mathcal{X}_{\mathrm{node}}\big)^{n}\times\big(\mathcal{X}_{\mathrm{edge}}\big)^{\binom{n}{2}}$ to represent undirected graphs. A graph $x\in \mathcal{X}$ contains a set of $n$ nodes $\mathcal{V}=\{1,\dots,n\}$ and edges $\mathcal{E}$ defining the connectivity between these nodes. Each node $i\in\mathcal{V}$ has a node class $x_i \in \mathcal{X}_{\mathrm{node}}$ (e.g.\ an atom type), and each edge $(i,j)\in\mathcal{E}$ has an edge class $x_{ij}\in\mathcal{X}_{\mathrm{edge}}$ (e.g.\ a bond type).
Note that $\mathcal{X}_{\mathrm{edge}}$ includes $\varnothing$ denoting a ``no-edge'' class. % We can obtain the edges from the edge features as
% $
% \mathcal{E}=\{(i,j): 1\le i<j\le n,\; x_{ij}\neq \varnothing\}.
% $
The graph $x:=\big(x_\mathcal{V},x_\mathcal{E}\big)$ consists of $x_\mathcal{V}$ and $x_\mathcal{E}$ denoting categorical node and edge features, respectively, and are defined as
\begin{equation}
x_\mathcal{V}  :=[x_i]_{i=1}^n\in(\mathcal{X}_{\mathrm{node}})^n;\quad
x_\mathcal{E}  :=[x_{ij}]_{ i<j}\in\big(\mathcal{X}_{\mathrm{edge}}\big)^{\binom{n}{2}}\nonumber
\end{equation}
% We consider undirected graphs, so that $x_{ji}=x_{ij}$.
We embed the node and edge categorical feature vectors $x_\mathcal{V}$ and $x_\mathcal{E}$ using a real-valued one-hot encoding and treat them as continuous features using
\[
\phi_{\mathcal{V}} :\mathcal{X}_{\mathrm{node}}\to\mathbb{R}^{l_{\mathrm{node}}};\quad
\phi_{\mathcal{E}} :\mathcal{X}_{\mathrm{edge}}\to\mathbb{R}^{l_{\mathrm{edge}}}
\]
Where $l_{\mathrm{node}}=|\mathcal{X}_{\mathrm{node}}|$ and $l_{\mathrm{edge}}=|\mathcal{X}_{\mathrm{edge}}|$ denotes number of node classes and edge classes, respectively. We denote the real-valued node $\hat{x}_{\mathcal{V}}\in\mathbb{R}^{n l_{\mathrm{node}}}$ and edge $\hat{x}_{\mathcal{E}}\in\mathbb{R}^{\binom{n}{2}l_{\mathrm{edge}}}$ features and the corresponding graph representation 
$\hat{x}$ as
% \begin{equation}
% \hat{x}_{\mathcal{V}}:=[\phi_{\mathcal{V}}(x_i)]_{i=1}^n; \quad \hat{x}_{\mathcal{E}}:=[\phi_{\mathcal{E}}(x_{ij})]_{ i<j};\quad \hat{x}:=\big[\hat{x}_{\mathcal{V}}, \hat{x}_{\mathcal{E}}\big]. \nonumber
% \end{equation}
\begin{equation}
\hat{x}_\mathcal{V}:=\oplus_{i=1}^n \phi_\mathcal{V}(x_i);\quad
\hat{x}_\mathcal{E}:=\oplus_{i<j}\phi_\mathcal{E}(x_{ij});\quad
\hat{x}:=\hat{x}_\mathcal{V}\oplus \hat{x}_\mathcal{E}. \nonumber
\end{equation}
where $\oplus$ denotes the element-wise concatenation operation.

\subsection{JKO-Style Variational Objective for Discrete Probability Spaces}
We introduce a JKO-style objective adapted to discrete graph configurations. Our goal is twofold: i) We preserve the Gibbs equilibrium associated with the learned energy $V_\theta$ and ii) remain well-defined on a finite state space where sampling proceeds via discrete edits rather than continuous trajectories. Hence, we avoid time-indexed dynamics, which are cumbersome for discrete jumps and often dominated by 'do-nothing moves' (self-loops or near-identity edits arising from invalid/high-cost proposals under graph constraints), by using an energy-based switch $\epsilon(x)$ and a coupling-based transport term defined over valid graphs. The switch is binary: $\epsilon(x)=0$ in the transport regime and $\epsilon(x)=1$ in the mixing regime; in our sampler it is triggered once the chain either reaches the expected energy level for graphs with that node count or hits a local minimum under greedy edits.

%his choice directly motivates the multi-site, geometry-aware sampler:


Given a distribution $\pi_k$, we consider the JKO update at $k+1$
\begin{equation}
\pi_{k+1}=\arg\min_{\pi}\,\mathcal{J}(\pi).
\end{equation}
with the objective
\begin{align}
\mathcal{J}(\pi)
&=
\frac{1}{2\eta}\,\inf_{\gamma\in\Pi(\pi_k,\pi)}\mathcal{A}(\gamma)\nonumber +\sum_{x\in\mathcal{X}} V_\theta(x)\,\pi(x)\nonumber\\
&\quad+\sum_{x\in\mathcal{X}} \epsilon(x)\,\pi(x)\log\pi(x).
\end{align}
We note that this formulation differs from the traditional JKO step. While the traditional JKO involves minimization of the kinetic term over couplings between $\pi_k$ and $\pi$, yielding $W_2^2$ in the continuous setting, we are operating in a discrete state space. Hence, we work with an action $\mathcal{A}(\gamma)$ induced by a cost function $c(x,y)$ defined between two graphs $x$ and $y$. \#FIXME A pointwise stationarity condition along paired graphs $(x,y)$ from an optimal coupling is, up to an additive constant,
\begin{equation}
\frac{1}{2\eta}\,c(x,y)
\;+\;V_\theta(x)
\;+\;\epsilon(x)\,\log\pi_{k+1}(x)
\;=\;\text{const}. \label{eq:relaxed_coupling}
\end{equation}
At the data distribution, the transport term vanishes, so with $\epsilon(x)=1$, this results in the Gibbs form
\begin{equation}
\pi_{\mathrm{eq}}(x)\propto \exp\left(-V_\theta(x)\right).
\end{equation}

However, further away from the data regime, in the off-equilibrium regime with $\epsilon(x)=0$, the functional can be analyzed by relaxing the kinetic term. We rely on a fixed coupling $\tilde{\gamma}$ between the source $\pi_0$ and the data distribution
$\pi_\mathrm{data}$ (e.g.\ minibatch OT), which induces a path between them and replaces the infimum
with $\mathcal{A}(\tilde{\gamma})$, yielding
\begin{equation}
\widetilde{\mathcal{J}}(\pi_{k+1};\tilde{\gamma})
=
\frac{1}{2\eta}\,\mathcal{A}(\tilde{\gamma})
\;+\sum_{x\in\mathcal{X}} V_\theta(x)\,\pi_{k+1}(x).
\end{equation}
By \eqref{eq:kinetic-action}--\eqref{eq:kinetic-action-bound}, $\mathcal{A}(\tilde{\gamma})$ upper-bounds
the OT discrepancy, and this relaxed term is tractable to compute. This motivates a \emph{transport-aligned
drift} based on the displacement between coupled samples; we use this to supervise
$-\nabla_{\hat{x}}V_\theta$ along the coupling paths. We evaluate gradients with respect to the
continuous embedding $\hat{x}$ introduced above, which makes the potential landscape well-defined away from
equilibrium and yields a training signal along the predefined coupling paths.
We want to point out that $\epsilon(x)$ is constant in each regime and does not require an
explicit time index, making it natural for jump processes and MCMC chains.





\subsection{Permutation-Invariant Minibatch OT}

While for the targeted potential $V_\theta(x)$, permutation invariance can be naturally enforced by construction as $V_\theta$ is produced by a permutation-equivariant graph transformer with an invariant readout to a scalar (see Appendix~\ref{app:perm-invariance} for details), so $V_\theta(\sigma \cdot x)=V_\theta(x)$ and the Gibbs distribution $\pi_\theta(x)\propto \exp(-V_\theta(x))$ are invariant. 
Here, $\sigma\in S_n: \mathcal{V}\rightarrow\mathcal{V}$ is a permutation that acts on node indices and induces a relabeling of the
graph: $(\sigma\cdot x)_i=x_{\sigma(i)}$ and $(\sigma\cdot x)_{ij}=x_{\sigma(i)\sigma(j)}$ and $S_n$ is the symmetry group of $n$ element.

To additionally enforce permutation invariance during the transport phase, we introduce a Gromov-Wasserstein (GW) cost function for pairing slection:


\paragraph{Local embedding discrepancy and fused GW cost.}
We measure local discrepancies in the embedding by
\begin{equation}\label{eq:loc-cost}
c_{\mathrm{loc}}(x,y)
\;\coloneqq\;
\lambda_\mathcal{V}\|\hat{x}_{\mathcal{V}}-\hat{y}_{\mathcal{V}}\|^2_2
+
\lambda_\mathcal{E}\|\hat{x}_{\mathcal{E}}-\hat{y}_{\mathcal{E}}\|^2_2,
\end{equation}
with $\lambda_\mathcal{V},\lambda_\mathcal{E}>0$ scaling node and edge contributions. To compare graphs in a permutation-invariant way,
we lift this local geometry to a fused discrepancy by optimizing over node relabelings:
\begin{equation}\label{eq:fgw-cost}
c_{\mathrm{FGW}}(x,y)
=
\min_{\sigma\in S_n}
\left\{
c_{\mathrm{loc}}(x,\sigma\cdot y)
\right\}.
\end{equation}
This quantity is used for minibatch pairings and transport supervision.

%In the sampler, we avoid re-solving for $\sigma$ at every step and instead use $c_{\mathrm{loc}}$ as a local surrogate for small edits.


\paragraph{Minibatch couplings and transport-aligned supervision.}
Given minibatches from a source $\pi_0$ and the data $\pi_\mathrm{data}$, we compute a permutation-invariant minibatch matching using an approximation to $c_{\mathrm{FGW}}$, obtaining a coupling
$\tilde{\gamma}$ and a pairing map $T$ that assigns each data graph $x^\mathrm{data}$ to a source
graph $y^0:=T(x^\mathrm{data})$.\\
Exact evaluation of $c_{\mathrm{FGW}}$ is NP-hard for graph matching or quadratic assignment and thus computationally infeasible. In molecule-like settings, we use node-matching alignments (with light
edge tie-breakers), while for a uniform source we use fast histogram signatures; see
Appendix~\ref{app:matching}. Since we transport from $\pi_0$ toward $\pi_\mathrm{data}$, these
pairings provide off-manifold supervision: the displacement
$v=\hat{x}^\mathrm{data}-\hat{y}^0$ defines a transport-aligned drift, which
we use to find $-\nabla_{\hat{x}}V_\theta$ via training objective
(Section~\ref{sec:training}).


\subsection{Proposal Generation during Transport and Mixing}

\paragraph{Local cost for proposals.}
To generate proposals, we propose \emph{local edits} of the current graph. We can thus use the local surrogate $c_{\mathrm{loc}}$ from \eqref{eq:loc-cost}, which
serves as a local approximation of the fused GW costs when edits are small. Therefore, we are not required to resolve for $\sigma$ at each step to minimize over permutations. \\
We use two proposal mechanisms, separated by
regime:  a deterministic greedy descent in
the transport regime and a stochastic geometry-aware kernel in the mixing regime (at the data distribution)

\paragraph{Transport-regime greedy proposal.}\label{par:transport-greedy}
In the transport regime, we use a deterministic greedy proposal: for each site $s$, we choose the locally best edit
$c_s^\star$ and its energy change $\Delta_s$, then apply the $N$ most negative edits in parallel. The MCMC
chain stops (energy-based switch) when no negative $\Delta_s$ remain or when $V_\theta(x)$ drops below a
reference energy level for the given node count, $\mathbb{E}_{\pi_\mathrm{data}}[V_\theta(x)\mid n]$, which we
estimate and cache during training. This update is accepted with probability $1$.


\paragraph{Mixing-regime stochastic proposal.}
We allow \emph{multi-site updates} that may change any subset of nodes and edges in a single step.
This proposal is inspired by the transport-aligned drift: taking a directional derivative of the
relaxed objective in Eq. \ref{eq:relaxed_coupling} yields
\[
(\hat{y}-\hat{x})^\top\big(\frac{1}{2\eta}\nabla_{\hat{x}} c_{\mathrm{loc}}(\hat{x},\hat{y})
-\nabla_{\hat{x}}V_\theta(\hat{x})\big) = 0
\], whose terms appear in the kernel below. The
exponential/softmax form supplies an exploration (entropic)
component, and the resulting kernel can be viewed as a \emph{linearized locally balanced proposal}; related
discrete-gradient samplers appear in~\citep{grathwohl2021oops,zhang2022langevin,zanella2020informed}.
Here we work in the continuous embedding, so displacements and gradients are taken with respect to
\(\hat{x}\) and \(\hat{y}\). Define the linearized energy change
\(\Delta V_\theta(x\to y)\approx V_\theta(y)-V_\theta(x)\) by
\(\Delta V_\theta(x\to y)\coloneqq(\hat{y}-\hat{x})^\top\big(-\nabla_{\hat{x}} V_\theta(\hat{x})\big)\).
We then use a gradient-informed transition kernel guided by the first-order optimality conditions of the
regularized objective. This is related to locally balanced proposals through this linearization and avoids
full energy differences in high dimension:
\begin{equation}\label{eq:loc-kernel}
\begin{aligned}
q_\theta(x\to y)
&\propto
\exp\Big(
(\hat{y}-\hat{x})^\top \big(-\beta\nabla_{\hat{x}} V_\theta(\hat{x})\big)\\
&\quad+\lambda_c\,(\hat{y}-\hat{x})^\top \nabla_{\hat{x}} c_{\mathrm{loc}}(\hat{x},\hat{y})
\Big)
\end{aligned}
\end{equation}
The explicit form for the local discrepancy in \eqref{eq:loc-cost} is:
\begin{equation}\label{eq:loc-kernel-explicit}
\begin{aligned}
q_\theta(x\to y)
&\propto
\exp\Bigg(
(\hat{y}-\hat{x})^\top \big(-\beta\nabla_{\hat{x}} V_\theta(\hat{x})\big)\\
&-
\lambda_\mathcal{V}\|\hat{y}_\mathcal{V}-\hat{x}_\mathcal{V}\|^2_2
-\lambda_\mathcal{E}\|\hat{y}_\mathcal{E}-\hat{x}_\mathcal{E}\|^2_2
\Bigg)
\end{aligned}
\end{equation}
In practice we calibrate $\beta$, $\lambda_\mathcal{V}$, and $\lambda_\mathcal{E}$ by maximizing relative likelihood of
generated samples under the learned energy.

\paragraph{Initializations and proposal schedules.}
We use two initialization regimes with aligned proposal schedules:
\begin{itemize}[leftmargin=*]
\item Noise initialization: sample the node count from the empirical histogram, then draw node/edge
types uniformly (uniform noise) or from empirical marginals (marginal noise). These start far from
the data manifold, so we run transport with greedy proposals until the energy-based switch (see
\Cref{par:transport-greedy}), then enter MH mixing with the proposal in
\eqref{eq:loc-kernel-explicit} and acceptance in \eqref{eq:mh-accept}.
\item Data initialization: draw graphs from the training set (novelty starts at zero by
construction). We skip transport and start directly in mixing with the same proposal
\eqref{eq:loc-kernel-explicit}, using a higher initial proposal temperature that is annealed before
fixed-temperature mixing (Appendix~\ref{app:sampling-details}).
\end{itemize}
All initializations start near-zero $V.U.N.$ (valid-unique-novel). Figure~\ref{fig:moses-energy-and-metrics}
summarizes the resulting energy trajectories under noise vs.\ data initialization and the
corresponding proposal regimes.

\paragraph{Mixing-regime acceptance.}
In the mixing regime, we accept the stochastic proposal with Metropolis--Hastings probability:
\begin{equation}\label{eq:mh-accept}
\begin{aligned}
\alpha(x,y)
&=\min\left(1,\frac{\pi_\theta(y)\,q_\theta(y\to x)}{\pi_\theta(x)\,q_\theta(x\to y)}\right),\\
\end{aligned}
\end{equation}
with $\pi_\theta(x)\propto \exp\left(-V_\theta(x)\right)$. This choice
enforces detailed balance:
\begin{equation}\label{eq:mh-detailed-balance}
\pi_\theta(x)\,q_\theta(x\to y)\,\alpha(x,y)
=
\pi_\theta(y)\,q_\theta(y\to x)\,\alpha(y,x),
\end{equation}
so $\pi_\theta$ is stationary for the mixing dynamics~\citep{metropolis1953equation,hastings1970monte}.
The locality term ties the sampler directly to the same quadratic form induced by the graph cost, while
dropping the permutation search is justified by the locality of edits under a fixed labeling.
Implementation details of the proposal kernel (factorized logits, two-temperature MH, and do-nothing
resampling) are provided in Appendix~\ref{app:sampling-details}.
